Final Project: Hangman

Project Description

Building upon the prototype from the midterm project, we chose to make an
even cleaner hangman game with added features for the player to interact with.

Project Team

  • Karen Lee,link: https://people.ucsc.edu/~klee57/art101/finalProject/hangman.html
  • Avery Lavender,link: https://people.ucsc.edu/~alwelsh/art101/finalProject/hangman.html

Design Considerations

To create a clean and cohesive look to the webpage, we used the same color palette
and font on the splash and game page. And because this is a game, we chose an arcade
style font to give it a more interesting aesthetic. The buttons on the game page
are all centered to allow for easier player navigation.

Challenges

The biggest challenge came when we were trying to find a color palette that had
different hues and tones while still being cohesive. Another tedious challenge
that we faced was placing the hangman figure parts in the correct position, which
we solved through trial and error.

Technology & Recognition

In addition to the string arrays, global variables, and helper functions that were
present in the prototype, we also introduced the window.onload event handler to
load everything to the page upon starting.
